So, how my final solution should look like?

You just need to wrap your code into function expression and return the result of your computations.

Here is the whole solution:

def middle(text: str) -> str:

    ind, rem = divmod(len(text), 2)
    result = text[ind+rem-1: ind+1]
        
    return result

Recursively create new strings without first and last characters from the text until its length become one or two. 

def middle(text: str) -> str:

    return text if len(text) < 3 else middle(text[1:-1])

This short expression uses a lot of instruments: lambda expression (anonymous function), walrus operator :=, which allows to define variable with initial value in first usage, integer division operator // and ~ operator.

If your l is 2 for example, you actually take the third symbol from the beginning of the string. ~ gives you the actual third character from the end of the string. For the string with odd number of symbols it will be the same symbol. Read more in this explanation. 

middle = lambda text: text[~(l:=(len(text)//2)): l+1]
